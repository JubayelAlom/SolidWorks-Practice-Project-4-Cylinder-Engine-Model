# 🔧 4-Cylinder Engine Assembly – SolidWorks Practice Project

## 📌 Project Overview

This project is a 4-cylinder internal combustion engine assembly modeled in SolidWorks as part of improving my CAD and mechanical design skills.

The objective of this project was to strengthen my understanding of mechanical components, assembly relationships, and parametric design principles by building a functional inline 4-cylinder engine configuration from individual parts.

---

## 🎯 Learning Objectives

Through this project, I focused on improving:

- Part Modeling  
- Assembly Design  
- Mates & Motion Relationships  
- Parametric Design Approach  
- Mechanical Component Detailing  
- Understanding of Crank–Slider Mechanism  

---

## 🛠️ Components Modeled

The assembly includes:

- Pistons  
- Connecting Rods  
- Crankshaft  
- Engine Block  
- Cylinder Housing  
- Fasteners (basic structural elements)

Each component was modeled separately and later assembled using appropriate mates to simulate realistic mechanical motion.

## ⚙️ Technical Details

- **Software Used:** SolidWorks  
- **Configuration:** Inline 4-Cylinder  
- **Mechanism Type:** Crank-Slider Mechanism  
- **Design Method:** Fully Parametric Modeling  
- **Assembly Type:** Multi-part assembly with defined motion constraints  

The crankshaft rotation drives the reciprocating motion of the pistons through connecting rods, replicating the basic working principle of an internal combustion engine.
